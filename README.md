# DigiWallets
An iOS app showing a list of MOCKED digital wallets and also the transactions made for those digital wallets.

## Schema
- `CryptoLoader` is a macOS based framwork
- `APILayer` is a macOS based framwork
- `iOS` is a iOS based framework

## Schema and Target selection
- Select `My Mac` when running the test for `CryptoLoader`
- Select `My Mac` when running the test for `APILayer`
- Select any iOS simulator when running the test for `iOS`

Not selecting the proper test target will cause build error.
